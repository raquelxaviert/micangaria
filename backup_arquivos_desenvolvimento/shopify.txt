---

## 🧪 **Tutorial: rotina de edição da Dawn no VS Code**

---

### 🗂️ **1. Abrir a pasta do tema**

Abra o VS Code na pasta onde está o seu tema:

```powershell
cd C:\Users\quelx\Desktop\repos\micangueria\dawn
code .
```

> Isso abre o VS Code com a pasta do seu tema carregada.

---

### 💻 **2. Abrir o terminal interno do VS Code**

No VS Code:

* Atalho: \`Ctrl + \`\` (crase) — para abrir o terminal integrado
* Ou: Menu superior > Terminal > "Novo terminal"

---

### 🚀 **3. Iniciar o ambiente de desenvolvimento**

Com o terminal aberto, rode o seguinte comando:

```powershell
shopify theme dev --store 1rfmi1-r5.myshopify.com
```

Se você não estiver logado, ele:

* Mostra um código de verificação
* Abre o navegador automaticamente
* Solicita a senha da loja (URL: `/admin/online_store/preferences`)

Depois disso, ele mostra:

* Um link local: `http://127.0.0.1:9292`
* E um link Shopify:
  `https://1rfmi1-r5.myshopify.com/admin/themes/[id]/editor?hr=9292`

---

### ✏️ **4. Editar arquivos à vontade**

Você pode modificar qualquer um desses arquivos:

* `sections/` – estruturas das seções da página
* `templates/` – layouts (ex: `product.liquid`)
* `snippets/` – pedaços de código reutilizáveis
* `assets/` – CSS, JS, imagens
* `config/settings_schema.json` – define o que pode ser personalizado no editor da Shopify

⚡ Preview com hot reload no navegador:
Todas as alterações são atualizadas automaticamente em `http://127.0.0.1:9292`.

---

### 💾 **5. (Opcional) Salvar suas alterações no Git**

Se você estiver usando Git:

```bash
git add .
git commit -m "Atualizações no tema"
git push origin main
```

---

### 📤 **6. (Opcional) Enviar para a loja Shopify**

Quando quiser criar um rascunho do seu tema na loja:

```powershell
shopify theme push --unpublished
```

Você pode então ativar ou editar esse tema direto no admin da Shopify.

---

## 🧠 Dica extra: Automatize com um script

Se quiser, posso te criar um script `.ps1` (PowerShell) que:

* Abre o VS Code
* Abre o terminal
* Roda o `shopify theme dev` automaticamente
